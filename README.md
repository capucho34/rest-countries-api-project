REST Countries API – Country Info Viewer

Displays information about countries and their neighbors. You can select a country to see detailed info on a separate page. Data is fetched from the REST Countries API.

Features

View all countries

Search countries by name

Filter by region

Country detail page with info and neighbors

Technologies

HTML

CSS

JavaScript (fetch API)

What I learned

Working with APIs and handling JSON

Dynamic page updates with JavaScript

Creating interactive elements (search, filter, neighbor links)
